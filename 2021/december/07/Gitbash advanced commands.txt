1. git status -s (dəyişiklikləri qısa formatda görmək)
2. git diff --staged (staging area'dakı edilən dəyişiklikləri göstərir diff = different)
3. git log (commit history'ni göstərir)
loga baxanda Head bizim hal-hazırda olduğumuz branchın ən son commitini təmsil edir.
4. git log --reverse (--reverse bizə historyni ilk commitdən son commitə göstərir)
5. git log --oneline (--oneline bizə historydəki commitlər barədə short info verir)
6. git show 052fb5e (commit ID'yə görə həmin commitin dəyişikliyini göstərir)
7. git show HEAD (son commit'dəki dəyişiklikləri göstərir)
8. git show HEAD~1 (son commit'dən bir commit əvvəlki commitin dəyişikliklərini göstərir)
    git show HEAD~2 (son commit'dən iki  commit əvvəlki commitin dəyişikliklərini göstərir)
9. git show HEAD~1:filename.txt (son commitdən bir commit əvvəlki commitdə spesifik bir fayldakı dəyişikliyi göstərir)
10. git remote add origin https://github.com........ (remote origin əlavə edir)
11. git branch (olduğu branch'ı göstərir)
12. git branch -M main (main adlı branch yaradır və keçir)
13. git push -u origin main (localdakı reponu githuba main branchı yaradıb push edir)
