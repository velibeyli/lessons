git clone https://github.com/velibeyli/classes_and_methods.git

git pull (branchda olan deyisiklikleri locala cekir)

git push (localdaki deyisiklikleri brancha gonderir)

git add "folder or file name" (deyisikliyi tracked changes e elave edir)

git commit -m "Commit text here" (-m = message)

git status (localdaki deyisiklikleri son pull etdiyin versiya ile muqayise edir)

git log (log history)

Siralama:
git clone [link] (ilk defe reponu clone edende istifade olunur)

git pull (deyisiklik var mi yoxlanilir)

git add . (senin etdiyin deyisiklikleri elave edir tracked e)

git commit -m "My commit" (commit mesajini yaziriq push etmeden once)

git push
